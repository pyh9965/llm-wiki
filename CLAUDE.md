# 분양업무 Wiki - Claude 작업 지침

## 지식 저장 원칙
작업 결과물 중 재사용 가능한 지식은 wiki/에 저장.
법령은 korean-law-mcp 조회 후 wiki/법령/에 저장.
참조 우선순위: wiki/ 먼저 → MCP 조회 → 결과를 wiki/에 저장.

## 디렉토리 구조
```
wiki/
├── 분양/       # 분양 절차, 일정, 계약 관련 지식
├── 법령/       # 주택법, 분양가상한제, 청약 관련 법령
├── 단지분석/   # 단지별 분석 자료 및 비교
├── 고객상담/   # 상담 FAQ, 응대 스크립트
├── 마케팅/    # 마케팅 전략, 카피, 채널별 운영
└── 더희/      # 더희 관련 자료
schema/         # 문서 템플릿 및 스키마
raw/            # 원본 데이터 (gitignore 처리됨)
```

## 작업 규칙
- 새로운 법령/규정 확인 시 → `wiki/법령/`에 저장
- 단지 분석 완료 시 → `wiki/단지분석/`에 저장
- 고객 FAQ 업데이트 → `wiki/고객상담/`에 추가
- 마케팅 카피/전략 → `wiki/마케팅/`에 저장
- 파일명: 영문 또는 한글, 하이픈(-) 구분

## MCP 조회 규칙
1. 법령 관련 질문 → 먼저 `wiki/법령/` 확인
2. 없으면 → `korean-law-mcp`로 조회
3. 조회 결과 → `wiki/법령/`에 정리하여 저장

## 커밋 컨벤션
- `docs: 새 위키 문서 추가`
- `update: 기존 문서 업데이트`
- `feat: 새 기능/스키마 추가`
