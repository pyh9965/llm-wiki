# LLM Wiki - Claude 작업 지침

## 프로젝트 개요
LLM(대형 언어 모델)에 관한 지식을 정리하는 위키 레포지토리입니다.

## 디렉토리 구조
```
wiki/
├── models/      # LLM 모델별 정리 (GPT, Claude, Gemini 등)
├── concepts/    # 핵심 개념 (Attention, Transformer, RLHF 등)
├── papers/      # 주요 논문 정리
├── tools/       # 관련 도구 및 프레임워크
└── benchmarks/  # 벤치마크 및 평가 방법
```

## 작성 규칙
- 모든 문서는 한글로 작성 (영어 용어는 병기 가능)
- 각 파일은 마크다운(.md) 형식 사용
- 파일명은 영문 소문자, 하이픈(-) 구분
- 출처 및 참고 링크 반드시 포함

## 커밋 컨벤션
- `docs: 새 문서 추가`
- `update: 기존 문서 수정`
- `fix: 오류 수정`
