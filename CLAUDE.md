# LLM Wiki 운영 규칙

## Wiki 규칙
- 작업 결과물 중 재사용 가능한 지식은 wiki/ 폴더에 마크다운으로 저장하라
- 기존 위키 페이지가 있으면 업데이트하라 (새로 만들지 말고)
- 위키 페이지 상단에 항상 최종 업데이트 날짜를 기록하라
- 법령 정보는 korean-law-mcp로 조회 후 wiki/법령/ 에 저장
- 새 단지 분석 시 wiki/단지분석/단지분석-템플릿.md 형식을 따르라

## 참조 우선순위
1. wiki/ 폴더에서 기존 정리본 먼저 확인
2. 없으면 MCP 도구로 원본 조회
3. 조회 결과를 wiki/에 저장

## Lint 규칙
- 법령 정보는 3개월마다 최신 여부 확인
- 실거래가 데이터는 1개월마다 갱신
- _index.md는 하위 페이지 추가/삭제 시 자동 업데이트
