# 분양업무 Wiki

분양 실무에 필요한 지식을 체계적으로 정리한 위키입니다.

## 📁 구조

```
wiki/
├── 분양/       # 청약 절차, 용어사전, 특별공급 가이드
├── 법령/       # 주택법, 지방세법, 분양가상한제 등 법령 정리
├── 단지분석/   # 단지별 입지·상품·가격 분석
├── 고객상담/   # 청약 FAQ, 상담 스크립트
├── 마케팅/    # 마케팅 전략, 채널별 운영
└── 더희/      # 더희 단지 관련 자료
schema/         # 문서 템플릿 및 작성 규칙
raw/            # 원본 데이터 (PDF, CSV 등 / git 제외)
```

## 🔍 참조 우선순위

1. **wiki/ 먼저 확인** → 기존 정리된 내용 활용
2. **없으면 MCP 조회** → `korean-law-mcp` 등 활용
3. **조회 결과를 wiki/에 저장** → 다음에 재사용

## ✏️ 문서 추가 방법

1. 해당 카테고리 폴더에 `.md` 파일 생성
2. 카테고리 `_index.md` 목차에 링크 추가
3. 커밋: `git commit -m "docs: 문서명 추가"`
