# 청약 용어사전

## 자격 관련

| 용어 | 설명 |
|------|------|
| 무주택세대구성원 | 세대 전원이 주택을 소유하지 않은 세대의 구성원 |
| 세대구성원 | 주민등록등본에 함께 등재된 가족 |
| 1순위 | 가입 기간·납입 횟수 등 청약통장 요건 충족자 |
| 2순위 | 1순위 미충족자 |
| 투기과열지구 | 청약 경쟁률·집값 상승 등으로 국토부 장관이 지정한 지역 |
| 청약과열지역 | 청약 경쟁이 과열된 지역 (투기과열지구보다 완화된 기준) |

## 분양가 관련

| 용어 | 설명 |
|------|------|
| 분양가 상한제 | 민간택지에서도 분양가 상한을 규제하는 제도 |
| 기본형 건축비 | 국토부 고시 표준 건축비 |
| 택지비 | 토지 매입·조성 비용 |
| HUG 분양보증 | 주택도시보증공사의 분양보증 |

## 공급 관련

| 용어 | 설명 |
|------|------|
| 특별공급 | 정책 수혜자(신혼·다자녀·생애최초 등)를 위한 우선 공급 |
| 일반공급 | 특별공급 외 일반 청약자 대상 공급 |
| 예비입주자 | 당첨자 계약 포기 시 순서대로 계약 기회 부여 |
| 전용면적 | 벽체 안쪽 실제 사용 면적 (발코니 제외) |
| 공급면적 | 전용면적 + 주거공용면적 |
