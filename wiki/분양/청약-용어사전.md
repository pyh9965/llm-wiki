# 청약 용어 사전

## 주요 용어

| 용어 | 설명 |
|------|------|
| 공급면적 | 전용면적 + 주거공용면적 |
| 전용면적 | 실제 거주 공간 (욕실, 거실, 방 등) |
| 발코니 확장 | 서비스 면적을 내부 공간으로 편입 |
| 무순위 청약 | 미분양 세대 대상 청약 (줍줍) |
| 가점제 | 무주택기간·부양가족·청약통장 가입기간 합산 점수 |
| 추첨제 | 가점 없이 추첨으로 당첨자 선정 |
| 전매제한 | 당첨 후 일정 기간 분양권 매도 금지 |
| 중도금 대출 | 계약금 납부 후 입주 전 공사 기간 중 대출 |
| HUG 보증 | 주택도시보증공사 분양보증 |

## 청약통장 종류
- **주택청약종합저축**: 모든 주택 청약 가능 (현재 신규 가입 가능)
- **청약저축**: 국민주택 전용 (신규 가입 불가)
- **청약예금/부금**: 민영주택 전용 (신규 가입 불가)
