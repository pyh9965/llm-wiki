# 분양가 상한제 계산법

## 공식

```
분양가 상한 = 택지비 + 건축비
```

### 택지비
- 공공택지: 공급가격 (감정가 기준)
- 민간택지: 감정평가액 (2개 기관 평균)
- 가산비: 택지조성비, 간선시설 부담금 등

### 건축비
- 기본형 건축비 (국토부 고시, 반기별 변동)
- 가산비: 지하층, 고층화, 친환경 인증 등

## 기본형 건축비 (2025년 기준 예시)
| 구분 | 금액 (원/㎡) |
|------|------------|
| 전용 60㎡ 이하 | 별도 고시 |
| 전용 85㎡ 이하 | 별도 고시 |
| 전용 85㎡ 초과 | 별도 고시 |

> 실제 수치는 국토부 고시 최신본 확인 필요

## 포인트
- 분양가 상한제 적용 단지는 전매제한·실거주의무 병행
- 민간택지 분양가 상한제: 투기과열지구 내 적용
