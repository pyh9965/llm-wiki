# 주요 논문 정리

LLM 분야의 중요한 논문들을 정리합니다.

## 기초 논문
| 연도 | 제목 | 기관 | 핵심 기여 |
|------|------|------|-----------|
| 2017 | Attention is All You Need | Google | Transformer 아키텍처 제안 |
| 2018 | BERT | Google | 양방향 사전학습 모델 |
| 2020 | GPT-3 | OpenAI | 대규모 언어 모델의 few-shot 학습 |
| 2022 | InstructGPT | OpenAI | RLHF를 통한 지시 따르기 |
| 2023 | LLaMA | Meta | 오픈소스 효율적 LLM |

## 최신 연구 트렌드
- Mixture of Experts (MoE)
- Long Context 처리
- Multimodal LLM
- Efficient Fine-tuning (LoRA, QLoRA)
