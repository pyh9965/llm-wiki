# 서비스 진행 프로세스

## 분양대행 프로세스

```
1. 상담 및 계약
   └─ 사업 개요 파악, 계약 조건 협의

2. 사전 분석
   └─ 입지 분석, 시장 조사, 경쟁 단지 분석

3. 마케팅 기획
   └─ 타겟 설정, 채널 전략, 콘텐츠 계획

4. 분양 실행
   └─ 모델하우스 운영, 상담 진행, 청약 접수 지원

5. 계약 관리
   └─ 계약자 관리, 잔금·입주 안내

6. 사후 관리
   └─ AS 연계, 고객 만족도 조사
```

## 청약 컨설팅 프로세스

1. 고객 자격 확인 (무주택, 소득, 자산)
2. 적합 단지 추천
3. 청약 전략 수립
4. 청약 신청 지원
5. 당첨 후 계약 안내
