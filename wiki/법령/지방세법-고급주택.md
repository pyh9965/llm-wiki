# 고급주택 기준 (취득세 중과)

## 근거법령
지방세법 제13조 제5항 제3호 → 지방세법 시행령 제28조 제4항

## 고급주택 기준 (어느 하나 해당 + 시가표준액 9억 초과)

| 호 | 기준 | 적용 대상 |
|---|---|---|
| 1호 | 연면적 331㎡ 초과 | 단독/공동 |
| 2호 | 대지면적 662㎡ 초과 | 단독/공동 |
| 2호의2 | 엘리베이터 설치 (200kg 초과) | 단독만 |
| 3호 | 에스컬레이터 또는 67㎡이상 수영장 | 단독만 |
| 4호 | 공동주택 전용 245㎡ 초과 | 공동만 |

## 포인트
- 소형 엘리베이터(200kg 이하)는 고급주택 해당 안됨
- 시가표준액 9억 이하이면 1,2,2의2,4호는 해당 안됨
