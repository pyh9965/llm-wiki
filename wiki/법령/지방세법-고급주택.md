# 지방세법 - 고급주택 기준

> 출처: korean-law-mcp 조회 결과 | 최종 확인: 2026-04-05

## 고급주택 정의 (지방세법 시행령 제28조)

취득세 중과세 대상 고급주택 기준:

| 구분 | 기준 |
|------|------|
| 단독주택 | 건물 연면적 331㎡ 초과 + 취득가액 9억 원 초과 |
| 공동주택 | 전용면적 245㎡ 초과 + 취득가액 9억 원 초과 |
| 공통 | 엘리베이터·에스컬레이터·67㎡ 이상 수영장 보유 시 면적 무관 |

## 취득세 중과세율
- 일반: 1~3%
- 고급주택: **12%** (지방교육세 포함 시 13.4%)

## 주의사항
- 취득가액 기준은 실거래가 기준
- 분양가 + 발코니 확장비 + 옵션 비용 합산 검토 필요
- 법령 변경 시 korean-law-mcp 재조회 후 업데이트
