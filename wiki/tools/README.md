# LLM 관련 도구 및 프레임워크

## 개발 프레임워크
| 도구 | 용도 | 언어 |
|------|------|------|
| LangChain | LLM 앱 개발 파이프라인 | Python/JS |
| LlamaIndex | RAG 구축 | Python |
| Hugging Face Transformers | 모델 로드 및 파인튜닝 | Python |
| vLLM | 고성능 LLM 서빙 | Python |
| Ollama | 로컬 LLM 실행 | Go |
