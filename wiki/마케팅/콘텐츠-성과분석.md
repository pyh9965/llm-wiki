# 콘텐츠 성과 분석

## 분석 지표 (KPI)

| 지표 | 설명 | 목표치 |
|------|------|-------|
| 도달 | 콘텐츠 노출 수 | - |
| 클릭률(CTR) | 클릭수 / 노출수 | 3% 이상 |
| 상담 전환율 | 상담 신청 / 클릭 | 5% 이상 |
| 계약 전환율 | 계약 / 상담 | 10% 이상 |

## 채널별 특성

| 채널 | 주요 타겟 | 콘텐츠 형식 |
|------|---------|-----------|
| 인스타그램 | 30대 실수요자 | 카드뉴스, 릴스 |
| 유튜브 | 40대 투자자 | 단지 소개 영상 |
| 네이버 블로그 | 검색 유입 | 상세 분석 글 |
| 카카오톡 채널 | 기존 고객 | 공지, 이벤트 |
